为了方便查阅，这里记录几个常用的功能选项。

## 查看已安装的Packages
使用`pip list`已安装的python packages:
```
pip list

--------------
Package         Version
--------------- ------------
catkin          0.7.18
catkin-pkg      1.0.0
cv-bridge       1.13.0.post1
distro          1.8.0
docutils        0.20.1
genmsg          0.5.12
...
```
